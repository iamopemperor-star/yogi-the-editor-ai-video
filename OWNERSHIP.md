# Yogi The Editor — AI Video Generator

## Ownership Declaration

**Creator:** Yogi The Editor
**Project:** AI Video Generator (Multi-AI Pipeline)
**Created:** March 2026
**First Deployment:** https://video-editor-junior-web.vercel.app

## What This Project Does
- Takes a video brief from the user
- Uses Gemini 2.5 Flash as AI prompt engineer
- Generates cinematic video via Runway Gen-4.5
- Streams real-time progress via SSE
- Delivers downloadable MP4

## Tech Stack
- Next.js 16 (App Router)
- Google Gemini 2.5 Flash API
- Runway Gen-4.5 API
- Vercel deployment
- TypeScript + Tailwind CSS
